### Pré-requisitos
- **Node** com versão superior ou igual que 16.15.0
- **Yarn** com versão superior ou igual que 2.2.0

## Instalando as dependências:

```
$ yarn
```
## Executando o projeto

### Produção

```
yarn dev
```
